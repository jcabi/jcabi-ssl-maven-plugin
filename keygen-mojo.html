jcabi-ssl:keygen

Full name:

com.jcabi:jcabi-ssl-maven-plugin:0.14.0:keygen

Description:

Generate SSL keystore and configure in JVM.

Attributes:

  • Requires a Maven project to be executed.
  • Since version: 0.5.
  • Binds by default to the lifecycle phase: initialize.

Optional Parameters

Name Type Since Description
<cacerts> File 0.5 Name of cacerts.jks file.
Default value is: ${project.build.directory}/cacerts.jks.
<keystore> File 0.5 Name of keystore.jks file.
Default value is: ${project.build.directory}/keystore.jks.
<skip> boolean 0.5 Shall we skip execution?
