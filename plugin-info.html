Plugin Details

This report describes goals, parameters details, requirements and sample usage of this plugin.

Goals

Goals available for this plugin:

Goal Description
jcabi-ssl:help Display help information on jcabi-ssl-maven-plugin.
Call mvn jcabi-ssl:help -Ddetail=true -Dgoal=<goal-name> to display parameter details.
jcabi-ssl:keygen Generate SSL keystore and configure in JVM.

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven 3.9.14
JDK 1.8

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>com.jcabi</groupId>
          <artifactId>jcabi-ssl-maven-plugin</artifactId>
          <version>0.15.0</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>com.jcabi</groupId>
        <artifactId>jcabi-ssl-maven-plugin</artifactId>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>

For more information, see "Guide to Configuring Plug-ins"
